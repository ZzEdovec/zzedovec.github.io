ZeScreenshot уважает вашу конфиденциальность. Он не обрабатывает и не хранит ваши скриншоты нигде более вашей локальной машины.

ZeScreenshot не имеет встроенного кода, позволяющего скачивать и/или устанавливать стороннее ПО. ZeScreenshot расчитан только на обновления самого себя.

ZeScreenshot контролирует нажатия клавиш клавиатуры, но не более чем для обеспечения работоспособности горячих клавиш.

ZeScreenshot не имеет встройной аналитики и/или рекламы. Вы можете сделать добровольное пожертвование для поддержания разработки.

ZeScreenshot может считывать и записывать информацию на ваш диск. Это нужно для обеспечения работы функций программы.

ZeScreenshot может соединяться с Интернетом без вашего ведома если у вас включена автоматическая проверка обновлений.

ZeScreenshot использует встроенное программное обеспечение ОС Microsoft Windows для обеспечения работоспособности функции редактирования скриншотов.
