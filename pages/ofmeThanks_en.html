Thank you for downloading OnlineFix Launcher for Linux!

If you cannot run the executable file, make it executable - right-click on the downloaded file, open Properties, and in the Permissions tab, allow the file to be executed as a program.

Or open a terminal in the folder with the downloaded installer and execute:

chmod +x onlinefix_launcher_installer && ./onlinefix_launcher_installer

The launcher requires the following dependencies: protontricks, ffmpeg, and 7zip.

You can find instructions for installing dependencies in the documentation on GitHub.
