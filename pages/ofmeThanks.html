Спасибо за загрузку OnlineFix Launcher для Linux!

Если не получается запустить исполняемый файл, сделайте его исполняемым - нажмите правой кнопкой мыши по скачанному файлу, откройте Свойства и во вкладке Права разрешите выполнение файла как программы

Или откройте терминал в папке со скачанным установщиком и выполните:

chmod +x onlinefix_launcher_installer && ./onlinefix_launcher_installer

Для работы лаунчера требуются следующие зависимости: protontricks, ffmpeg и 7zip. Установите их, используя команды ниже для вашего дистрибутива:

Ubuntu и производные:

sudo apt install protontricks ffmpeg p7zip-full

Solus:

sudo eopkg install protontricks ffmpeg p7zip

Fedora:

sudo dnf install protontricks ffmpeg p7zip

Arch Linux:

sudo pacman -S protontricks ffmpeg 7zip
