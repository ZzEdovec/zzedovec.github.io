Для запуска любого установщика программ ZeD Team на Linux вам необходима Java 8 или OpenJRE 8

Здесь рассмотрен вариант запуска через OpenJRE (с openjfx), т.к. он интегрируется в систему самостоятельно.

Установка OpenJRE

Установка OpenJRE можно выполнить через центр приложений, доступный в вашей системе, или через терминал:

Для установки через терминал на Ubuntu и основанных на ней дистрибутивах введите:

sudo apt-get install openjdk-8-jre openjfx

для Arch Linux и основанных на нём дистрибутивах:
sudo pacman -S jre8-openjdk java8-openjfx

Установка OpenJRE для jar файлов по умолчанию и запуск установщика

Для GNOME (Nautilus):

Нажмите правой кнопкой мыши по установщику и выберите Открыть в другом приложении. OpenJRE будет доступна в появившимся списке. Выберите её и нажмите на кнопку Установить по умолчанию. После этого вы сможете запустить наш установщик двойным щелчком мыши (или одним, в зависимости от ваших настроек)

Для KDE, XFCE, Cinnamon и прочих сред рабочего стола, использующих файловые менеджеры Dolphin, Thunar или Nemo по умолчанию

Нажмите правой кнопкой мыши по установщику и наведите мышь на Открыть с помощью, а затем нажмите кнопку Другое приложение. В появившемся окне выберите строку для ввода (в некоторых случаях вам надо будет развернуть пункт Использовать команду) и напишите в неё java -jar, после чего поставьте галочку на Использовать по умолчанию для этого типа файлов. Теперь вы можете запустить установщик нужной вам программы от ZeD Team двойным щелчком мыши (или одним, в зависимости от ваших настроек).

Не удаляйте OpenJRE в дальнейшем! Все программы ZeD Team для Linux работают на Java. В случае удаления они перестанут запускаться и работать.
