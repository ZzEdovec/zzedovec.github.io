Это ZeNetwork. Он соединяет все сервисы ZeD Team, Voshod Development и ZzEdovec в одну сеть, которая в перспективе должна стать экосистемой.
На данный момент ZeNetwork находится лишь на стадии разработки и тестирования,
но мы уже можем сообщить о том, какие возможности он будет представлять:
  • Удобный лаунчер для всех современных операционных систем, поддерживающих Oracle Java или OpenJdk
  • Ресурсы для сообщества
  • Поддержка для продуктов компаний и разработчиков, которые входят в сеть ZeNetwork
  • Веб-сайт, на котором можно будет найти всю необходимую информацию


Мы запустим ZeNetwork к 2023 году.
